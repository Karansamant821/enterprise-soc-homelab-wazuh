# enterprise-soc-homelab-wazuh
Enterprise SOC Home Lab using Wazuh SIEM with Windows and Linux endpoints, attack simulation, threat detection, vulnerability monitoring, and MITRE ATTACK mapping.

---> 
This project demonstrates the implementation of a Security Operations Center (SOC) monitoring environment using the Wazuh security platform.

The lab simulates a small enterprise environment where multiple endpoints send security telemetry to a centralised SIEM platform for threat detection, vulnerability monitoring, and security analysis.
